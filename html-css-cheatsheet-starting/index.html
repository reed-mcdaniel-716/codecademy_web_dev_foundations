CSS Cheat Codes

Fonts and Text

  • font-family: represents what is typically refered to simply as the "font". Examples include "Time New Roman", san-serif, and Georgia.

Box Sizing

  • margin: the measurement between the border of an element and the next element placed on top of, underneath, or besides it (outside).
  • padding: the measurement between the border of an element and the content of the element itself (inside).

Colors

  • named color: a predefineds set of colors given descriptive names. Examples include "cornflowerblue" and "firebrick".
  • hex(): hexidecimal (16 digit) representation of a color.
  • rgb(): a triplet of values 0-255 representing the amount of red, green, and blue to make up a color respectively.
